# Commands
* Cloning: ```git clone git@github.com:lotteunckell/MotherNature.git ```
* Saving Progress:
  1. Making a new Branch: ```git checkout -b <new_name>```
  2. Adding: ```git add -a```
  3. Committing: ```git commit -m "<message>"```
  4. Pushing to GitHub: ```git push origin <branch_name>```
